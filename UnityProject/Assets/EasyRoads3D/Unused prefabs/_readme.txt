This folder can be used to store connection prefabs that you will not use in your scenes.

These prefabs are located in:

/Assets/EasyRoads3D/Resources/dynamic prefabs/
/Assets/EasyRoads3D/Resources/custom prefabs/

It will clean up the connections tab in the Inspector

These prefabs can always be moved back to /EasyRoads3D/Resources/dynamic prefabs/ and /EasyRoads3D/Resources/custom prefabs/
